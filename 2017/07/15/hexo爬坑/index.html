乘着暑假的时间来搞一些事情，Blog这个东西其实一直想做了。之前由于个人的知识储备比较少，解决问题的经验也不够丰富（其实是懒，滑稽)。一直没有能够完成。暑假的话正好时间相对比较充裕，本人也已经有了一些折腾的经验，就用两天时间到腾出了这个Blog。这个博客采用了Hexo框架，整体安装按照官方的文档来说应该是比较容易的。
首先安装git
sudo apt install git
然后去nodejs官网去下载安装nodejs。
最后使用
npm install hexo-cli -g
来安装Hexo。

但是第二天就傻眼了，我再次运行hexo却提示command not found,非常坑爹，于是乎继续上网寻求解决方案，找了好久在这里(假装有链接)找到了有效的解决方案：先安装nvm，再安装hexo。顺序很重要，所以要想解决问题应该要先sudo apt remove nodejs，再删除位于root文件夹的npm相关文件夹，安装nvm，再利用nvm install安装nodejs，最后再安装hexo。
经过测试，这样安装的hexo在重新启动之后不会遇到command not found的情况了，如果以后再遇到其它坑，再来补坑吧，最后按例放一张老婆的照片。

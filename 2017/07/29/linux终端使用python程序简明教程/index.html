linux终端使用python程序简明教程

Posted on 2017-07-29 | Edited on 2018-05-18

笔者目前使用的发行版是 openSUSE 42.3，在此之前使用的是 Debian 9。不同的发行版所打包的程序不尽相同，之前使用 Debian 的时候很喜欢一个可以直接在终端使用的有道词典客户端ydcv。这个程序之前 Debian_cn已经有人打包好了，拿来下载即用，非常舒服，可是 openSUSE 暂时没有人去完成这项工作。为了使用它，笔者查阅了资料，简要了解了一下如何在 Linux 终端使用 Python 程序。

首先做好准备工作，安装Python及相关工具。

接着从作者的Github上获取源码包并将其移动到本地文件夹。

接着添加alias让使用更便捷。笔者使用的是 zsh，所以需要修改.zshrc文件。
sudo vim ~/.zshrc
在末端加上
alias ydcv='/usr/local/ydcv/ydcv.py’，设置就已经成功了。但是为了避免重启终端，可以使用
source ~/.zshrc命令，接下来就可以愉快的使用命令行版的有道词典了。
效果如图所示：

PS

对于终端进行的许多操作并不会立即生效，这时候需要使用诸如source ~/.zshrc这样的操作让使用更加便捷畅快。

使用hexo+Github Pages搭建个人博客

Posted on 2017-07-29 | Edited on 2018-05-18 | Views:

笔者之前写过一篇Hexo爬坑，里面记录了使用 Hexo 搭建博客时遇到的一些问题，对于搭建的过程并没有详细描述，现在通过这篇文章来记录一下使用 Hexo 和 Github Pages 搭建博客的过程，权当是一篇教程了。

  • 首先需要注册一个 Github 账号，然后 选择 Create a new repository ，仓库的名字是 yourgithubid.github.io，例如笔者的仓库名字就是 tonyable.github.io 。
  • 笔者之前使用了直接安装nodejs的方法，遇到了各种各样奇葩的问题，索性直接使用 nvm 来安装。
    使用命令
    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
    安装 nvm ，之后重启终端或者使用命令
    source ~/.zshrc
    ，之后键入 nvm 检查是否安装正确。
  • 键入命令
    nvm install 8
    安装node。
  • 安装Hexo。键入
    npm install hexo-cli -g，待读条完成后键入 hexo 出现如下界面证明安装完成。

将最后的 Deployment修改为

  • 如果使用 Git 方式，还需要安装
    npm install hexo-deployer-git，并对git进行如下设置

将自己的 SSH 公钥上传至 Github ，之后在终端输入
ssh -T git@github.io
检测是否建立成功。
vim _config.yml,
将repo更改为：
repo: git@github.com:username/username.github.io。
在博客目录下运行

至此一个简易的博客算是搭建完成了，后续的更换主题等操作并不在此次教程范围内，以后有时间再写吧。

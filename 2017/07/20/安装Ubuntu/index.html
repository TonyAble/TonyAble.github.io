x6ti 安装Ubuntu

Posted on 2017-07-20 | Edited on 2018-05-18

出于性能的考虑（其实是为了玩游戏233 ），去年入手了这台游戏本机械革命x6ti。然而作为一名计算机专业的学生，有不少时候需要用到Linux系统，但是用虚拟机总是各种各样的不爽，遂准备装一个 win10 +Ubuntu 双系统的机器。由于NVIDIA及其坑爹，这台机器又不能够在BIOS里直接禁用掉核心显卡，所以安装还是费了一番波折。

笔者的计算机配置：

  • CPU : intel i7 6700HQ
  • 内存 8G
  • 显卡：NVIDIA GTX 965m 4G显存版

具体的安装步骤如下：

  • 先使用 rufus 或者软碟通等工具将下载好的镜像烧录进U盘，注意需要此操作会删除U盘内的所有数据，故需要先做好数据备份。 然后右击文件管理器中的”此电脑“，选择”管理“，用磁盘管理的功能划分出一块适合大小的空间（约50G）用来安装Ubuntu。

  • 重启计算机，快速按住F10，选择从优盘启动，回车。

  • 按e进入编辑界面，于quiet splash 后先打一个空格，再打“acpi=off”（无需引号），按下F10，开始安装过程。

  • 分区过程中推荐使用16GB挂载根目录（“/”），350m挂载/boot目录，其他全部挂载/home。由于此计算机配有×GB内存，所以个人认为并不需要设置交换分区。

  • 安装过程完成后使用Ubuntu 自带的驱动管理功能安装上NVIDIA的闭源驱动以及下面一个驱动，随后使用命令

    sudo vim /etc/default/grub 编辑grub文件。将acpi=off改成acpi=on，再于其后加上pci=nomsi ,按esc后按下：wq，回车保存并退出。执行命令sudo update-grub 后重启电脑，Ubuntu已经安装在电脑中，可以愉快地食用了。

    PS

    笔者目前使用的发行版并不是Ubuntu，而是Debian 9，相对来说安装更为方便，但是显卡驱动的问题较难解决，故不推荐新手使用。

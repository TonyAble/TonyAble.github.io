汇编学习

Posted on 2017-09-24 | Edited on 2018-05-18 | Views:

近期刚刚开始学习汇编，通过博客记录一些东西，巩固知识点。

  • CPU字长
    CPU在单位时间内能一次处理的二进制位数.该指标反映出CPU内部运算处理的速度.
  • 位宽
    CPU外部总线与外部设备之间一次能够传送的数据位.
  • x位CPU
    通常用字长和位宽来称呼CPU.
    举例
  • intel阵营
    • 8086 16位 CPU (字长16位,位宽16位)
    • 386 准32位 CPU (字长32位,位宽16位)
    • Pentium 超32位 CPU (字长32位,位宽64位)
  • 同期竞争品
    • Zilog Z8000 (16位CPU,对标8086)
    • Motorola 6800 (8位CPU,对标8080)
    • Am386 (80386的打鸡血克隆版,农企强无敌!)

#####ISA(Instruction Set Architecture)

  • 即指令集架构
  • 一个处理器支持的指令和指令的字节级编码称为它的ISA.
  • 不同的处理器”家族”系列,有不同的ISA.
  • 同一系列里也有不同类型的处理器.
ps

虽然每个厂商制造的处理器性能和复杂性不断提高，但是不同的类型在ISA级别上都保持着兼容.

x86

Intel现在称其指令集为IA32指令集,也就是“Intel 32位体系结构”（Intel Architecture 32-bit）.

IA32指令集，又被俗称为x86。这个处理器系列也俗称x86系列CPU.

IA32架构属于复杂指令集,随着80386的上市被公诸于世,用于取代 x86_16位架构(曾被用于8086,80186,80286等).

通用寄存器

8086CPU中,寄存器AX,BX,CX, DX通常用于存放一般性数据,称为通用寄存器.每个寄存器均为16位.

32位CPU中,通用寄存器为EAX,EBX,ECX,EDX.

64位 CPU中,通用寄存器为RAX,RBX,RCX,RDX.

  • 以AX为例，可将AH和AL作为8位寄存器单独使用

    AH和AL存放的无符号数据范围: 0 ~ 2^8-1

  • 一个字由两个字节组成,可以存在一个16位寄存器中.

    • 字的高8位 → 存放于通用寄存器的高8位寄存器.
    • 字的低8位 → 存放于通用寄存器的低8位寄存器.

####不同的定址模式

对于8086来说,段地址 × 16 + 偏移地址 = 物理地址.
